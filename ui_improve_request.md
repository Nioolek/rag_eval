# UI 迭代优化请求

## 问题
当前的 RAG 评测系统 UI 界面太丑了，需要迭代优化。

## 要求

1. **主题和配色**
   - 使用更现代化的主题
   - 更好的配色方案（专业、清爽）
   - 统一的色彩系统

2. **布局优化**
   - 改进整体布局结构
   - 更好的视觉层次
   - 合理的间距和留白

3. **组件美化**
   - 表格样式优化
   - 按钮设计改进
   - 卡片组件美化
   - 表单元素优化

4. **视觉元素**
   - 添加合适的图标
   - 添加视觉装饰元素
   - 改进图标和 emoji 使用

5. **Header/Footer**
   - 设计更专业的页眉
   - 改进页脚信息展示

6. **响应式设计**
   - 确保不同屏幕尺寸下的良好显示

## 文件位置
- `src/ui/app.py` - 主应用入口
- `src/ui/components/` - 各个 tab 组件

请分析现有代码并进行改进。
